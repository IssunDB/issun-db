## issundb-retrieval

Hybrid retrieval for [IssunDB](https://github.com/IssunDB/issun-db), an embedded graph database written in Rust.

This crate combines graph traversal, vector hits, text hits, and property filters with score fusion and subgraph materialization.
It is an internal crate; applications should depend on the [`issundb`](https://crates.io/crates/issundb) facade instead.
