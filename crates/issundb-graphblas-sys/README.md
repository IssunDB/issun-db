## issundb-graphblas-sys

Raw FFI bindings to the [SuiteSparse:GraphBLAS](https://github.com/DrTimothyAldenDavis/GraphBLAS).

The build script compiles GraphBLAS from source as a position-independent static library.
Bindings are generated by `bindgen`.
Building this crate needs CMake and Clang to be installed.

Most users should use the safe wrapper ([`issundb-graphblas`](https://crates.io/crates/issundb-graphblas)) instead of using this crate directly.
