## issundb-graphblas-sys

Raw FFI bindings to the Apache-2.0 [SuiteSparse:GraphBLAS](https://github.com/DrTimothyAldenDavis/GraphBLAS) C library, built
for [IssunDB](https://github.com/IssunDB/issun-db).

The build script compiles GraphBLAS from source as a position-independent static library, resolving the source from the `ISSUNDB_GRAPHBLAS_SRC`
override, the in-repo git submodule, or a pinned, checksum-verified tarball. Bindings are generated by `bindgen`. Building requires cmake and clang.

Most users should use the safe wrapper, [`issundb-graphblas`](https://crates.io/crates/issundb-graphblas), instead.
